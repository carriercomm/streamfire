Streamfire

Campfire rooms streamed to your terminal

Download as .zip Download as .tar.gz View on GitHub

Simple Campfire client that streams the room to terminal.

Installation

npm install -g streamfire

Then make sure you have a configuration file setup as ~/.streamfire/config.json. Here is a minimal configuration:

{
    "domain": "YOURDOMAIN.campfirenow.com",
    "apiToken": "YOURTOKEN"
}

Running

streamfire

This will give the full help and a list of rooms available to join under the domain specified in your configuration file.

Once you have joined a room hit F1 or F12 to see a list of available hotkeys.
